# VBA サンプル


- ## excel-sample.bas
  - Excelで動作


- ## excel-column.bas
  - Excelの列の操作


- ## excel-row.bas
  - Excelの行の操作

- ## excel-to-dict.bas
  - Excelの表をDictionaryに変換


- ## word-sample.bas
  - Wordと連携


- ## pp-sample.bas
  - PowerPointと連携


- ## pivot-sample.bas
  - ピボットテーブルを作成
  - pivot-sample-data.tsv

- ## csv-sample.bas
  - CSVファイルと連携
